# Scripture Daily — Flutter Bible App

A production-quality Bible app built with Flutter. Features splash screen, full auth flow (Google, Apple, Email), Bible reader with 7+ versions and 4 languages, daily prayer journal, bookmarks, and more.

## 🚀 Getting Started

### 1. Prerequisites

- Flutter SDK `>=3.0.0`
- Dart `>=3.0.0`
- Android Studio or VS Code with Flutter plugin
- Xcode (for iOS builds)

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
# Debug mode
flutter run

# Release build (Android)
flutter build apk --release

# Release build (iOS)
flutter build ipa --release
```

## 🔐 Authentication Setup

The app currently uses **demo mode** (any credentials work). To enable real auth:

### Firebase Setup

1. Create a project at [console.firebase.google.com](https://console.firebase.google.com)
2. Add Android and iOS apps to the project
3. Download `google-services.json` → place in `android/app/`
4. Download `GoogleService-Info.plist` → place in `ios/Runner/`
5. Add `firebase_core` and `firebase_auth` init in `main.dart`:

```dart
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart'; // generated by FlutterFire CLI

await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
```

6. Run FlutterFire CLI:
```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

### Google Sign-In

1. Enable Google in Firebase Auth → Sign-in methods
2. For Android: SHA-1 fingerprint must be added in Firebase console
3. For iOS: Add `REVERSED_CLIENT_ID` to `Info.plist` CFBundleURLSchemes
4. Uncomment the Google Sign-In code in `auth_provider.dart`

### Apple Sign-In (iOS only)

1. Enable in Firebase Auth → Sign-in methods
2. In Xcode: Signing & Capabilities → + Capability → Sign In with Apple
3. Uncomment the Apple Sign-In code in `auth_provider.dart`

## 📖 Bible API

The app uses [bible-api.com](https://bible-api.com) — a free, open Bible API.

### Supported Translations

| ID | Name | Language |
|----|------|----------|
| `kjv` | King James Version | English |
| `web` | World English Bible | English |
| `bbe` | Bible in Basic English | English |
| `asv` | American Standard Version | English |
| `clementine` | Latin Vulgate | Latin |
| `almeida` | João Ferreira de Almeida | Portuguese |
| `rccv` | Romanian Corrected Cornilescu | Romanian |

> **Note:** For production, consider [API.Bible](https://scripture.api.bible/) which offers 2,000+ translations including Yoruba (YCB), Hausa, Igbo, and many more African languages.

## 🌍 Adding More Languages / Versions

To add more Bible versions, edit `lib/models/bible_data.dart`:

```dart
BibleVersion(
  id: 'ycb',            // translation ID from API.Bible
  name: 'Yoruba Contemporary Bible',
  abbreviation: 'YCB',
  language: 'Yorùbá',
  languageCode: 'yo',
),
```

## 🛠 Customization

### Change app name
- Android: `android/app/src/main/AndroidManifest.xml` → `android:label`
- iOS: `ios/Runner/Info.plist` → `CFBundleDisplayName`

### Change colors
Edit `lib/theme/app_theme.dart`:
```dart
static const Color primary = Color(0xFF1A3A5C);  // Change this
static const Color accent  = Color(0xFFD4A843);  // And this
```

### Add Yoruba / Hausa / Igbo Bibles
Sign up for a free API key at [api.bible](https://scripture.api.bible) and swap the `BibleService` base URL and add language entries to `BibleData.versions`.
